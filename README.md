## 项目概述
**二维码/条形码解码工具**是一款功能强大、界面友好的桌面应用程序，支持多种格式的二维码和条形码解码。无论您是普通用户还是专业人士，都能轻松识别各种编码信息。

### 核心特点
- 支持12种主流编码格式（包括QR码、Data Matrix、PDF417等）
- 历史记录管理功能
- 剪贴板图片直接识别
- 数据库备份与导出
- 美观的马卡龙色系界面

## 快速入门
### 基础使用三步走
1. **加载图片**：点击"加载图片"按钮或直接拖放图片到窗口
2. **解码**：点击"解码二维码/条形码"按钮
3. **查看结果**：右侧文本框显示解码内容

### 快捷键
- `Ctrl+V`：从剪贴板粘贴图片
- `Ctrl+C`：复制解码结果

## 专业功能解析
### 支持的编码格式
| 编码类型 | 技术名称 | 典型应用 |
|---------|----------|----------|
| QRCODE | 二维码 | 移动支付、网页链接 |
| CODE128 | Code 128 | 物流包裹、医疗设备 |
| DATA MATRIX | Data Matrix | 电子元件标识 |
| PDF417 | PDF417 | 身份证件、登机牌 |

### 高级功能
1. **数据库优化**：定期执行可提升查询性能
2. **批量导出**：支持CSV/JSON格式，便于数据分析
3. **剪贴板识别**：实时监控剪贴板图片变化

### 技术实现
- 基于OpenCV的图像处理
- PyZbar解码核心
- SQLite数据存储
- PyQt5图形界面

## 常见问题解答
### Q1: 为什么无法识别某些二维码？
A：可能原因：
- 图片模糊或反光
- 使用了不支持的编码格式
- 二维码尺寸过小

### Q2: 历史记录存储在哪里？
A：程序目录下的`qrcode_history.db`文件中

### Q3: 如何迁移数据到新电脑？
A：备份`qrcode_history.db`文件，复制到新电脑相同位置

### Q4: 识别率不高怎么办？
A：尝试：
- 使用更高清的图片
- 调整图片角度
- 确保环境光线充足
