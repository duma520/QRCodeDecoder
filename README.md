## 项目简介

**二维码/条形码解码工具**是一款功能强大的跨平台应用程序，专为解码各种二维码和条形码而设计。无论您是普通用户还是专业人士，都能轻松使用本工具完成解码任务。

### 适用人群
- **普通用户**：快速解码商品条形码、微信二维码等
- **商务人士**：处理会议二维码、名片信息
- **教育工作者**：解码教学材料中的二维码内容
- **IT专业人士**：调试和测试二维码生成效果
- **零售业从业者**：快速检查商品条形码信息
- **仓储物流人员**：解码物流单号条形码

---

## 功能概述

### 核心功能
1. **多种格式支持**：可解码QR码、Code 128、EAN-13等12种常见格式
2. **多种输入方式**：
   - 从文件加载图片
   - 从剪贴板直接粘贴图片
3. **历史记录管理**：
   - 自动保存解码记录
   - 收藏重要记录
   - 批量删除记录
4. **数据导出**：支持CSV和JSON格式导出
5. **数据库管理**：备份、优化数据库

### 支持的解码类型
| 类型名称 | 常见用途 | 示例 |
|---------|---------|------|
| QRCODE | 网站链接、联系方式 | 微信二维码 |
| CODE128 | 物流、仓储 | 快递单号 |
| EAN13 | 商品零售 | 超市商品条码 |
| PDF417 | 证件、票据 | 身份证二维码 |
| DATA MATRIX | 工业标记 | 电子元件标识 |

## 安装指南

### 系统要求
- 操作系统：Windows 7+/macOS 10.12+/Linux
- Python 3.6+
- 至少200MB磁盘空间

### 安装步骤
1. **基础安装**：
   ```bash
   pip install PyQt5 pyzbar opencv-python numpy
   ```
2. **下载程序**：
   - 从GitHub发布页面下载最新版本
   - 或克隆仓库：
     ```bash
     git clone https://github.com/duma520/QRCodeDecoder.git
     ```

### 快速启动
双击`QRCodeDecoder.py`或运行：
```bash
python QRCodeDecoder.py
```

## 使用说明

### 基本操作流程
1. **加载图片**：
   - 点击"加载图片"按钮选择文件
   - 或使用`Ctrl+V`粘贴剪贴板图片
2. **解码**：点击"解码"按钮
3. **查看结果**：结果将显示在右侧面板
4. **复制结果**：点击"复制"按钮

### 详细操作指南

#### 1. 解码商品条形码（适合普通用户）
1. 用手机拍摄商品条形码
2. 在电脑上打开本软件
3. 点击"加载图片"选择照片
4. 点击"解码"按钮
5. 查看商品编码信息

#### 2. 批量处理二维码（适合商务人士）
1. 准备多个二维码图片
2. 依次加载并解码
3. 使用"导出CSV"功能保存所有结果
4. 在Excel中分析数据

#### 3. 调试二维码（适合开发人员）
1. 生成测试用二维码
2. 使用本工具解码
3. 检查解码结果与预期是否一致
4. 调整二维码生成参数

## 技术细节

### 解码原理
1. 图像预处理：转换为灰度图
2. 定位标记识别：寻找二维码/条形码特征点
3. 数据提取：解析二进制数据
4. 编码转换：转换为可读文本

### 支持的编码格式
- UTF-8
- ASCII
- ISO-8859-1
- 自动检测编码

### 性能指标
- 平均解码时间：<500ms
- 最大图片尺寸：8192x8192像素
- 最小识别尺寸：50x50像素（二维码）

## 常见问题

### Q1: 为什么无法解码我的二维码？
A: 可能原因：
- 图片模糊
- 二维码损坏
- 不支持的二维码类型

### Q2: 历史记录存储在哪里？
A: 程序目录下的`qrcode_history.db`文件

### Q3: 如何迁移历史记录到新电脑？
A: 备份数据库文件并复制到新电脑的相同位置

### Q4: 能解码彩色二维码吗？
A: 可以，但黑白对比度高的二维码识别率更高

### Q5: 支持批量解码吗？
A: 目前需要手动逐个解码，但可以批量导出结果
